## Mask3D (2023)

> Schult et al. | ICML 2023 | Query-based 3D 实例分割

Mask2Former 的 3D 延伸，将 Query-based 范式引入点云实例分割。

### 与 Mask2Former 的对应

| 2D (Mask2Former) | 3D (Mask3D) |
|------------------|-------------|
| Image | Point Cloud |
| Backbone (Swin/ResNet) | Sparse 3D U-Net (或 PTv3) |
| Pixel Decoder | Voxel/Point Feature Pyramid |
| Masked Attention | 3D Masked Attention (kNN / Voxel Query) |

### 意义

证明了 Query-based 方法在稀疏、无序的 3D 数据上同样优于传统聚类方法 (Clustering-based)，彻底解决了"聚类半径难调"的痛点。
