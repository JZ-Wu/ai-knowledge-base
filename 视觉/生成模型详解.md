# 生成模型详解：VAE 与 Diffusion

生成模型的核心目标：学习数据分布 $p(x)$，从中采样生成新数据。本文覆盖两个最重要的生成范式：**VAE（变分自编码器）** 和 **Diffusion（扩散模型）**。

> 参考：[苗思奇 - 变分自编码器 VAE](https://zhuanlan.zhihu.com/p/348498294) | [Lilian Weng - What are Diffusion Models?](https://lilianweng.github.io/posts/2021-07-11-diffusion-models/) | [Yang Song - Score-Based Generative Models](https://yang-song.net/blog/2021/score/)

## 一、从 Autoencoder 到 VAE

### 1.1 为什么 Autoencoder 不是生成模型

Autoencoder 学习编码器 $f$ 和解码器 $g$，使得 $g(f(x)) \approx x$。训练完之后，解码器 $g$ 看起来很诱人——它只需要一个低维向量 $z$ 就能输出一张图片。那我们能不能直接在 $z$ 空间里随机采样，然后用 $g$ 生成图片？

答案是：**不行**。绝大多数随机生成的 $z$ 只会产生无意义的噪声。

```
为什么？
  AE 没有对 z 的分布 p(z) 进行建模
  训练数据编码后，z 只分布在高维空间中极有限的几个区域
  z 空间绝大部分是"空白"，解码器从未见过这些区域的输入

  z 空间示意图：
  z₁ ●          ● z₃       ← 训练数据编码后落在这几个点附近
         ???                ← 这些区域解码器从未见过，输出垃圾
       ●
      z₂
```

**核心问题**：AE 只优化重建，不关心 $z$ 空间的结构。VAE 的思路就是：**显式对 $z$ 的分布进行建模**，强制 $z$ 服从一个已知分布（标准正态），使得空间中任意点都能解码出有意义的样本。

### 1.2 VAE 的生成视角

上一节说了 AE 不能生成，因为它没有对 $z$ 的分布建模。VAE 的核心思路是：**用概率语言重新定义整个生成过程**，把"生成一张图片"变成一个有明确概率含义的事情。

#### 生成过程的概率建模

假设世界上所有图片都是这样"生成"出来的：

```
Step 1:  先在某个低维空间随机抽一个"概念种子"   z ~ p(z) = N(0, I)
         （比如 z 可能编码了"橘色猫、立耳、侧脸"这些高层特征）

Step 2:  给定这个种子，通过一个参数为 θ 的生成器，   x ~ p_θ(x|z)
         把它"渲染"成一张具体的图片
```

这里 $z$ 是不可观测的隐变量（latent variable），$\theta$ 是 Decoder 的参数。

#### 什么是 $p_\theta(x)$？

$p_\theta(x)$ 是一个**概率密度函数**，表示参数为 $\theta$ 的模型认为数据 $x$ 附近的概率密度有多大。注意它不是"概率"——图片是连续数据，单个点没有概率，只有密度。唯一的约束是对所有可能的 $x$ 积分为 1：$\int p_\theta(x) dx = 1$。密度值本身可以大于 1。

直觉上：如果模型训好了，真实图片 $x$ 附近的密度应该很高（"这张猫图很合理"），而随机噪声附近的密度应该很低（"这不像是自然图片"）。训练目标就是调整 $\theta$，让真实数据所在区域的密度尽可能高。

#### 训练目标：最大化边际似然

我们希望模型"认可"训练数据——对于训练集中的每张图片 $x$，$p_\theta(x)$ 越大越好。数学上就是**最大化边际似然（marginal likelihood）**。

但 $p_\theta(x)$ 怎么算？$x$ 可以由**任何** $z$ 生成出来，所以需要把所有可能的 $z$ 的贡献加起来（积分）：

$$p_\theta(x) = \int p_\theta(x|z) \, p(z) \, dz$$

取对数：

$$\log p_\theta(x) = \log \int p_\theta(x|z) \, p(z) \, dz$$

```
直觉理解：
  问模型："你觉得这张猫图 x 出现的概率有多大？"
  模型回答："让我考虑所有可能的隐变量 z：
            z₁ 编码了橘猫特征 → 生成 x 的概率 p(x|z₁) 较高
            z₂ 编码了狗的特征 → 生成 x 的概率 p(x|z₂) 几乎为零
            z₃ 编码了黑猫特征 → 生成 x 的概率 p(x|z₃) 也很低
            ......
            把所有 z 的贡献积分起来，就是我认为 x 出现的总概率。"
```

这个积分看起来很美，但实际上**算不了**——$z$ 的维度太高，穷举所有可能的 $z$ 不现实。这就是下一节要解决的问题。

### 1.3 为什么直接优化不可行

为了算 $p_\theta(x)$，我们需要对 $z$ 积分。直觉上，可以从先验 $p(z) = \mathcal{N}(0, I)$ 中采样大量 $z$，然后蒙特卡洛估计：

$$p_\theta(x) \approx \frac{1}{L}\sum_{l=1}^{L} p_\theta(x | z^{(l)}), \quad z^{(l)} \sim \mathcal{N}(0, I)$$

**问题**：$z$ 空间维度高，而与某个 $x$ 真正相关的 $z$ 只占极小的区域。盲目从 $p(z)$ 中采样，绝大部分 $z$ 和 $x$ 毫无关系，$p_\theta(x|z)$ 几乎为零。要准确估计这个积分，需要天文数字的采样量。

**解决方案**：引入后验分布 $p_\theta(z|x)$——如果我们知道了 $x$ 对应的 $z$ 大概在哪，就可以只在那个小区域采样，效率极高。但真实后验 $p_\theta(z|x) = \frac{p_\theta(x|z)p(z)}{p_\theta(x)}$ 分母又是那个算不了的积分……

这就引出了**变分推断**：用一个可学习的近似分布 $q_\phi(z|x)$（编码器）去逼近真实后验。

### 1.4 Decoder 的设计

Decoder 的职责是拟合似然 $p_\theta(x|z)$。给定一个 $z$，Decoder 输出 $x$ 服从的分布的参数。

假设给定 $z$ 后，$x$ 的每个维度独立，服从高斯分布：

$$p_\theta(x|z) = \mathcal{N}(x; \mu_\theta'(z), \sigma'^2 I)$$

```
直觉举例：
  数据集 X = 所有猫的图片
  采样 z ~ N(0,I) → 某个 z 可能代表"橘色、立耳猫"这一类
  Decoder 输出 μ'(z) = 这类猫图片像素值的均值
  从 N(μ', σ'²I) 采样 → 得到一张具体的橘色立耳猫图片
```

**实践中的简化**：通常把 $\sigma'^2$ 设为固定超参数，Decoder 只需要输出 $\mu_\theta'(z)$，直接作为重建图片。

### 1.5 Encoder 的设计

Encoder 拟合近似后验 $q_\phi(z|x)$，给定数据点 $x$，输出 $z$ 应该服从什么分布。

由于先验和似然都是高斯分布，真实后验 $p_\theta(z|x)$ 也是高斯分布。因此我们假设：

$$q_\phi(z|x) = \mathcal{N}(z; \mu_\phi(x), \sigma_\phi^2(x) I)$$

Encoder 网络输入 $x$，输出 $\mu_\phi(x)$ 和 $\log \sigma_\phi^2(x)$（输出 $\log \sigma^2$ 而非 $\sigma$，保证方差为正且数值稳定）。

### 1.6 ELBO 推导

既然 $\log p_\theta(x)$ 不好直接算，我们找它的**下界**（Evidence Lower Bound, ELBO）。

$$\log p_\theta(x) = \log p_\theta(x) \cdot \underbrace{\int q_\phi(z|x) dz}_{=1}$$

$$= \int q_\phi(z|x) \log p_\theta(x) \, dz$$

$$= \int q_\phi(z|x) \log \frac{p_\theta(x,z)}{p_\theta(z|x)} \, dz \quad \text{(贝叶斯定理: } p_\theta(x) = \frac{p_\theta(x,z)}{p_\theta(z|x)}\text{)}$$

$$= \int q_\phi(z|x) \log \frac{p_\theta(x,z)}{q_\phi(z|x)} \cdot \frac{q_\phi(z|x)}{p_\theta(z|x)} \, dz$$

$$= \underbrace{\int q_\phi(z|x) \log \frac{p_\theta(x,z)}{q_\phi(z|x)} \, dz}_{\text{ELBO}} + \underbrace{D_{KL}(q_\phi(z|x) \| p_\theta(z|x))}_{\geq 0}$$

由于 KL 散度非负：

$$\boxed{\log p_\theta(x) \geq \text{ELBO} = \mathbb{E}_{q_\phi(z|x)} \left[\log \frac{p_\theta(x,z)}{q_\phi(z|x)}\right]}$$

**这个式子的美妙之处**：最大化 ELBO 同时做到了两件事：
1. **最大化 $\log p_\theta(x)$**（数据似然，生成模型的目标）
2. **最小化 $D_{KL}(q_\phi \| p_\theta(z|x))$**（让近似后验逼近真实后验）

进一步展开 ELBO：

$$\text{ELBO} = \mathbb{E}_{q_\phi(z|x)} [\log p_\theta(x,z) - \log q_\phi(z|x)]$$

$$= \mathbb{E}_{q_\phi(z|x)} [\log p_\theta(x|z) + \log p(z) - \log q_\phi(z|x)]$$

$$= \underbrace{\mathbb{E}_{q_\phi(z|x)} [\log p_\theta(x|z)]}_{\text{重建项 (Reconstruction)}} - \underbrace{D_{KL}(q_\phi(z|x) \| p(z))}_{\text{正则项 (Latent Loss)}}$$

| 项 | 含义 | 直觉 |
|---|---|---|
| 重建项 | 从编码的 $z$ 能多好地重建 $x$ | Encoder-Decoder 配合好，信息别丢 |
| KL 正则项 | $q_\phi(z \mid x)$ 要接近先验 $\mathcal{N}(0,I)$ | **强制潜在空间有好的结构**，让任意采样的 $z$ 都有意义 |

### 1.7 重参数化技巧（Reparameterization Trick）

VAE 架构中有一步"从 $q_\phi(z|x)$ 中采样 $z$"，但**采样操作不可导**，梯度无法反传给 $\phi$。

**技巧**：把随机性移到一个与参数无关的噪声变量 $\epsilon$ 上：

$$z = \mu_\phi(x) + \sigma_\phi(x) \odot \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

```
原来:  φ → q_φ(z|x) → 采样 z (不可导)  → Decoder → Loss
                         ✗ 梯度断了

重参数化后:
       φ → μ_φ(x), σ_φ(x)  →  z = μ + σ⊙ε  → Decoder → Loss
            可导        可导       可导（ε 当作常数）
                                    ↑
                               ε ~ N(0,I) 外部输入
```

$z$ 的分布没变，还是 $\mathcal{N}(\mu, \sigma^2 I)$，但现在 $z$ 是关于 $\mu, \sigma$ 的确定性函数，梯度可以正常反传。

### 1.8 KL 散度的解析解

$q_\phi(z|x) = \mathcal{N}(\mu, \sigma^2 I)$ 与 $p(z) = \mathcal{N}(0, I)$ 的 KL 散度有闭式解。

先看一维情况（$q = \mathcal{N}(\mu, \sigma^2)$，$p = \mathcal{N}(0, 1)$）：

$$D_{KL}(q \| p) = \int q(z) \log \frac{q(z)}{p(z)} dz = \frac{1}{2}(\mu^2 + \sigma^2 - \log \sigma^2 - 1)$$

推广到 $d$ 维（各维度独立）：

$$D_{KL}(q \| p) = \frac{1}{2} \sum_{j=1}^{d} \left(\mu_j^2 + \sigma_j^2 - \log \sigma_j^2 - 1\right)$$

### 1.9 完整损失函数

将重建项和 KL 项合并（取负号变为最小化）：

$$\mathcal{L}_{\text{VAE}} = \underbrace{-\mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)]}_{\text{重建损失}} + \underbrace{D_{KL}(q_\phi(z|x) \| p(z))}_{\text{KL 正则}}$$

**重建项展开**：假设 $p_\theta(x|z) = \mathcal{N}(x; \mu_\theta'(z), \sigma'^2 I)$，其中 $x$ 维度为 $D$：

$$-\log p_\theta(x|z) = \frac{D}{2}\log(2\pi\sigma'^2) + \frac{1}{2\sigma'^2}\|x - \mu_\theta'(z)\|^2$$

去掉常数项后，重建损失正比于 $\frac{1}{2\sigma'^2}\|x - \mu_\theta'(z)\|^2$。

实践中从 $q_\phi(z|x)$ 只采样一个 $z$（$L=1$），最终损失：

$$\mathcal{L} = \frac{1}{2\sigma'^2}\|x - \hat{x}\|^2 + \frac{1}{2}\sum_{j=1}^{d}(\mu_j^2 + \sigma_j^2 - \log \sigma_j^2 - 1)$$

其中 $\hat{x} = \mu_\theta'(z)$ 是 Decoder 输出。

### 1.10 实现中的常见陷阱

**关于重建损失中的 $\sigma'^2$**：

很多 PyTorch 实现直接用 `F.mse_loss(x_recon, x, reduction='mean')` 计算重建损失。**这是错误的！**

```python
# ❌ 错误做法
recon_loss = F.mse_loss(x_recon, x, reduction='mean')  # 除以了 D

# ✅ 正确做法
recon_loss = F.mse_loss(x_recon, x, reduction='sum')   # 不除以 D
```

`F.mse_loss(..., reduction='mean')` 等价于 $\frac{1}{D}\sum_i(x_i - \hat{x}_i)^2$，多除了一个数据维度 $D$。

**为什么这很关键？** 这个 $D$ 不是无关紧要的常数——它实际上控制了**重建损失和 KL 正则之间的相对权重**。多除一个 $D$（对于 MNIST，$D=784$），等价于把 Decoder 似然分布的方差 $\sigma'^2$ 放大了 $D$ 倍。方差极大意味着模型认为"重建得不像也无所谓"，Decoder 就不会好好重建了。

```python
# 完整的正确实现
def vae_loss(x, x_recon, mu, log_var):
    # 重建损失：用 sum 而非 mean
    recon_loss = F.mse_loss(x_recon, x, reduction='sum')
    # KL 散度（解析解）
    kl_loss = 0.5 * torch.sum(mu.pow(2) + log_var.exp() - log_var - 1)
    return recon_loss + kl_loss
```

### 1.11 VAE 为什么生成模糊？

这是最经典的问题。主要有两种解释：

**解释一：直接用均值代替采样**

Decoder 本应输出的是 $x$ 服从的分布 $\mathcal{N}(\mu', \sigma'^2 I)$，然后从中采样。但实际中我们直接拿均值 $\mu'$ 当生成结果。均值天然是"平均"了所有可能性的，自然模糊。

**解释二（更本质）：高斯假设 + 有限的模型容量**

单个 $z$ 可能对应**多种**不同的 $x$（如不同姿态的猫），真实的 $p(x|z)$ 是多峰的。但我们假设它是单峰高斯，模型只能把这些不同的可能性**平均**成一张模糊图片来最小化 MSE。

```
真实情况:  z₁ 对应 {猫A, 猫B, 猫C}  → p(x|z₁) 是多峰的
高斯假设:  只能输出一个均值         → 平均了 A/B/C → 模糊

本质原因: 模型容量有限，学到的 z 无法完全把不同样本区分开
         + 高斯假设只允许单峰输出
         → 只好平均
```

### 1.12 VAE 的改进方向

| 问题 | 改进 | 代表 |
|---|---|---|
| 生成模糊 | 换更复杂的似然分布 / 加 GAN 判别器 | VAE-GAN |
| 后验坍缩（KL 项太强，$z$ 被忽略） | KL annealing（$\beta$-VAE：逐渐增大 KL 权重） | $\beta$-VAE |
| 高斯假设太简单 | 用 Normalizing Flows 增强后验表达力 | IAF-VAE |
| 连续空间不适合离散数据 | 向量量化，离散化潜在空间 | **VQ-VAE** |

### 1.13 Conditional VAE (CVAE)

原版 VAE 无法控制生成什么类别。CVAE 引入标签 $y$ 作为条件：

```
原版 VAE:
  Encoder:  q_φ(z|x)        Decoder:  p_θ(x|z)

CVAE:
  Encoder:  q_φ(z|x, y)     Decoder:  p_θ(x|z, y)

做法: 将 y (如 one-hot 类别向量) 拼接到 Encoder 和 Decoder 的输入中
     → 推理时: 指定 y="7" + 采样 z → 生成数字 7
```

### 1.14 VQ-VAE：离散潜在空间

VQ-VAE（van den Oord et al., 2017）用**向量量化**替代连续高斯分布，是 Stable Diffusion 中 VAE 组件的基础：

```
Encoder → 连续向量 z_e → 找 Codebook 中最近的向量 z_q → Decoder

Codebook: {e₁, e₂, ..., e_K}  (K 个可学习的嵌入向量)
z_q = e_k,  k = argmin_i ||z_e - e_i||

损失 = 重建损失 + ||sg[z_e] - e||²  + β||z_e - sg[e]||²
                    codebook loss      commitment loss
       (sg = stop gradient)
```

VQ-VAE 生成图像比连续 VAE 更清晰，因为离散 codebook 天然避免了"取均值"的模糊问题。

## 二、Diffusion Models（扩散模型）

### 2.1 核心思想

扩散模型的思路极其简洁：

```
前向过程：逐步往图片上加噪声，直到变成纯高斯噪声
反向过程：学一个神经网络，逐步从噪声中去噪，还原出图片

  x₀ (清晰图片)                                    x_T (纯噪声)
    →  加噪  →  加噪  →  加噪  →  ......  →  加噪  →
    ←  去噪  ←  去噪  ←  去噪  ←  ......  ←  去噪  ←
         ↑        ↑        ↑                    ↑
     神经网络  神经网络  神经网络             神经网络
     (共享参数，额外输入时间步 t)
```

### 2.2 前向过程（加噪）

给定干净图片 $x_0$，在每一步 $t$ 加一点高斯噪声：

$$q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} \, x_{t-1}, \, \beta_t I)$$

其中 $\beta_t \in (0, 1)$ 是噪声调度（noise schedule），控制每步加多少噪声。

**关键性质：可以一步到位**。定义 $\alpha_t = 1 - \beta_t$，$\bar{\alpha}_t = \prod_{i=1}^{t} \alpha_i$，则：

$$q(x_t | x_0) = \mathcal{N}(x_t; \sqrt{\bar{\alpha}_t} \, x_0, \, (1 - \bar{\alpha}_t) I)$$

即：

$$x_t = \sqrt{\bar{\alpha}_t} \, x_0 + \sqrt{1 - \bar{\alpha}_t} \, \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

**直觉**：$\bar{\alpha}_t$ 从接近 1 逐渐衰减到接近 0。$t$ 小时信号占主导；$t$ 大时噪声占主导。

```
t=0:     x₀ (原图)           ᾱ₀ ≈ 1.0
t=250:   x₂₅₀ (略有噪点)     ᾱ₂₅₀ ≈ 0.7
t=500:   x₅₀₀ (很模糊)       ᾱ₅₀₀ ≈ 0.3
t=750:   x₇₅₀ (几乎是噪声)   ᾱ₇₅₀ ≈ 0.05
t=1000:  x₁₀₀₀ (纯噪声)      ᾱ₁₀₀₀ ≈ 0.0
```

### 2.3 反向过程（去噪）

如果我们知道 $q(x_{t-1} | x_t)$，就能从噪声逐步恢复图片。但它需要知道整个数据分布，不可行。

**关键观察**：当我们额外知道 $x_0$ 时，$q(x_{t-1} | x_t, x_0)$ 是**精确可算的高斯分布**：

$$q(x_{t-1} | x_t, x_0) = \mathcal{N}(x_{t-1}; \tilde{\mu}_t, \tilde{\beta}_t I)$$

其中：

$$\tilde{\mu}_t = \frac{\sqrt{\bar{\alpha}_{t-1}} \beta_t}{1 - \bar{\alpha}_t} x_0 + \frac{\sqrt{\alpha_t}(1 - \bar{\alpha}_{t-1})}{1 - \bar{\alpha}_t} x_t, \qquad \tilde{\beta}_t = \frac{1 - \bar{\alpha}_{t-1}}{1 - \bar{\alpha}_t} \beta_t$$

训练时 $x_0$ 已知（就是训练数据），所以可以用这个精确后验来指导学习。

### 2.4 训练目标推导

用变分推断（和 VAE 思路相同），最大化数据似然的下界：

$$\log p_\theta(x_0) \geq -L_{\text{VLB}}$$

VLB 拆解为每一步的 KL 散度：

$$L_{\text{VLB}} = \underbrace{D_{KL}(q(x_T|x_0) \| p(x_T))}_{L_T \text{ (常数)}} + \sum_{t=2}^{T} \underbrace{D_{KL}(q(x_{t-1}|x_t, x_0) \| p_\theta(x_{t-1}|x_t))}_{L_{t-1}} + \underbrace{(-\log p_\theta(x_0|x_1))}_{L_0}$$

每个 $L_{t-1}$ 都是两个高斯分布的 KL 散度（闭式解），归结为让模型预测的均值 $\mu_\theta(x_t, t)$ 逼近真实的 $\tilde{\mu}_t$。

### 2.5 从预测均值到预测噪声

将 $x_0 = \frac{1}{\sqrt{\bar{\alpha}_t}}(x_t - \sqrt{1-\bar{\alpha}_t}\epsilon)$ 代入 $\tilde{\mu}_t$：

$$\tilde{\mu}_t = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}} \epsilon \right)$$

所以模型只要预测加入的噪声 $\epsilon$，就能算出均值。令 $\epsilon_\theta(x_t, t)$ 为噪声预测网络：

$$\mu_\theta(x_t, t) = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right)$$

### 2.6 简化损失函数（DDPM 的核心结论）

Ho et al. (2020) 发现，去掉 $L_{t-1}$ 中与 $t$ 相关的权重系数后，训练效果反而更好：

$$\boxed{L_{\text{simple}} = \mathbb{E}_{t, x_0, \epsilon} \left[ \| \epsilon - \epsilon_\theta(x_t, t) \|^2 \right]}$$

**DDPM 的训练目标**——简洁到令人惊讶：**随机选一个时间步，加噪声，让网络预测加了什么噪声**。

### 2.7 训练与采样算法

```
训练算法（极其简单）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
repeat:
    x₀ ~ 训练数据
    t  ~ Uniform({1, ..., T})
    ε  ~ N(0, I)
    x_t = √ᾱ_t · x₀ + √(1-ᾱ_t) · ε        # 一步加噪
    梯度下降: ||ε - ε_θ(x_t, t)||²           # 预测噪声
until 收敛

采样算法（逐步去噪）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
x_T ~ N(0, I)                                # 从纯噪声开始
for t = T, T-1, ..., 1:
    z ~ N(0, I)  if t > 1, else z = 0
    x_{t-1} = 1/√α_t · (x_t - β_t/√(1-ᾱ_t) · ε_θ(x_t, t)) + √β̃_t · z
return x₀
```

### 2.8 噪声调度（Noise Schedule）

| 类型 | 公式 | 特点 |
|------|------|------|
| Linear（DDPM） | $\beta_t$ 从 $10^{-4}$ 线性增到 $0.02$ | 简单，但末端信噪比下降太快 |
| Cosine（Improved DDPM） | $\bar{\alpha}_t = \frac{f(t)}{f(0)}$，$f(t) = \cos\left(\frac{t/T + s}{1+s} \cdot \frac{\pi}{2}\right)^2$ | 更平滑，图像质量更好 |

### 2.9 与 Score Matching 的统一

**Score function**：数据分布对数密度的梯度 $\nabla_x \log p(x)$，指向数据密度增大的方向。

DDPM 的噪声预测 $\epsilon_\theta$ 和 score 有直接关系：

$$\nabla_{x_t} \log q(x_t) = -\frac{\epsilon}{\sqrt{1 - \bar{\alpha}_t}} \quad \Rightarrow \quad s_\theta(x_t, t) = -\frac{\epsilon_\theta(x_t, t)}{\sqrt{1 - \bar{\alpha}_t}}$$

**预测噪声 = 预测 score 的负方向（缩放后）**。DDPM 和 Score-Based Models（NCSN）本质上在做同一件事。

Yang Song 等人用 **SDE（随机微分方程）** 统一了两个框架：

$$\text{前向 SDE: } dx = f(x,t)dt + g(t)dw$$

$$\text{反向 SDE: } dx = [f(x,t) - g^2(t)\nabla_x \log p_t(x)]dt + g(t)d\bar{w}$$

DDPM 和 NCSN 只是同一个连续时间 SDE 的不同离散化方式。

等价的**概率流 ODE**（确定性版本）：

$$dx = \left[f(x,t) - \frac{1}{2}g^2(t)\nabla_x \log p_t(x)\right]dt$$

这建立了扩散模型与 Normalizing Flows 之间的联系，可以精确计算 log-likelihood。

## 三、加速采样

DDPM 需要 $T=1000$ 步采样，太慢。以下是主要加速方法。

### 3.1 DDIM（Denoising Diffusion Implicit Models）

Song et al. (2020) 定义**非马尔可夫**的前向过程，使得可以跳步采样：

$$x_{t-1} = \sqrt{\bar{\alpha}_{t-1}} \underbrace{\left(\frac{x_t - \sqrt{1-\bar{\alpha}_t} \cdot \epsilon_\theta}{\sqrt{\bar{\alpha}_t}}\right)}_{\text{预测的 } x_0} + \sqrt{1 - \bar{\alpha}_{t-1} - \sigma_t^2} \cdot \epsilon_\theta + \sigma_t \cdot z$$

$\sigma_t = 0$ 时，采样变成**确定性的**，且可以大幅跳步：

```
DDPM:  x₁₀₀₀ → x₉₉₉ → x₉₉₈ → ... → x₁ → x₀   (1000 步)
DDIM:  x₁₀₀₀ → x₉₀₀ → x₈₀₀ → ... → x₁₀₀ → x₀  (10 步，质量略降)
```

### 3.2 Classifier-Free Guidance（CFG）

条件生成的标准方法。训练时以一定概率（如 10%）随机丢弃条件，推理时加权组合：

$$\bar{\epsilon}_\theta = (1 + w) \cdot \epsilon_\theta(x_t, t, c) - w \cdot \epsilon_\theta(x_t, t, \varnothing)$$

$w$ 是 guidance scale，控制 prompt 的影响强度。

```
w=0:    纯无条件生成
w=7.5:  平衡点（Stable Diffusion 默认）
w=20:   高度贴合 prompt，但过饱和、多样性低
```

### 3.3 Consistency Models

Song et al. (2023)：训练模型从任意噪声水平 $x_t$ **一步直接映射到** $x_0$，不需要逐步去噪。

## 四、Latent Diffusion Model (LDM) / Stable Diffusion

### 4.1 动机

直接在像素空间做扩散计算量巨大（$512 \times 512 \times 3 = 786\text{K}$ 维）。LDM 的解决方案：

```
像素空间扩散:  在 512×512×3 上做 T 步去噪   → 慢，贵
潜在空间扩散:  在 64×64×4 上做 T 步去噪     → 快 ~48 倍，质量几乎不变
```

### 4.2 架构

```
训练:
  图片 x (512×512×3)
    → VAE Encoder → z₀ (64×64×4)           ← 压缩到潜在空间
    → 前向扩散加噪 → z_t
    → U-Net 预测噪声 ε_θ(z_t, t, c)        ← 在潜在空间去噪
    → 简化 loss: ||ε - ε_θ||²

推理:
  z_T ~ N(0, I)  (64×64×4)
    → U-Net 逐步去噪（DDIM 加速）→ z₀
    → VAE Decoder → 图片 (512×512×3)        ← 解码回像素空间
```

### 4.3 条件注入

U-Net 通过 **Cross-Attention** 注入文本条件：

```
文本 "a cat sitting on a chair"
  → CLIP Text Encoder → 文本嵌入 (77×768)
  → 作为 Cross-Attention 的 K, V

U-Net 中间层特征 → 作为 Q
  → Cross-Attention(Q, K, V) → 融合文本信息的特征
```

### 4.4 VAE 在 Stable Diffusion 中的角色

Stable Diffusion 的 VAE（KL-regularized autoencoder）负责：

- **编码**：将 512×512 图片压缩为 64×64 的潜在表示（空间压缩 8 倍）
- **解码**：将去噪后的潜在表示还原为高分辨率图片

VAE 单独预训练，扩散过程中冻结不更新。这正是 VAE 和 Diffusion 两个范式的**联合**。

## 五、三大生成范式对比

| | VAE | GAN | Diffusion |
|---|---|---|---|
| **训练目标** | 最大化 ELBO（重建 + KL） | 极小极大博弈 | 预测噪声（简化 VLB） |
| **生成方式** | 采样 $z$ → 解码 | 采样 $z$ → Generator | 从噪声逐步去噪 |
| **训练稳定性** | 稳定 | 不稳定（模式坍缩） | 稳定 |
| **生成质量** | 模糊（MSE 偏好均值） | 锐利但多样性差 | **最高质量** |
| **采样速度** | 快（一步） | 快（一步） | 慢（需要多步） |
| **似然评估** | 有下界（ELBO） | 无 | 有下界（VLB） |
| **代表模型** | VQ-VAE, DALL-E 1 | StyleGAN, BigGAN | DDPM, Stable Diffusion, DALL-E 2/3 |

## 六、面试高频问题

> **Q: VAE 和 AE 的本质区别？**
> A: AE 只优化重建，$z$ 空间无结构，不能生成。VAE 通过 KL 正则强制 $z$ 服从 $\mathcal{N}(0,I)$，使得任意采样的 $z$ 都能解码出有意义的样本。ELBO = 重建项 - KL 项，两者 trade-off。

> **Q: 重参数化技巧解决了什么问题？**
> A: "从分布中采样"这个操作不可导，梯度无法反传给 Encoder 参数。把 $z = \mu + \sigma \odot \epsilon$（$\epsilon \sim \mathcal{N}(0,I)$）重写后，$z$ 变成关于 $\mu, \sigma$ 的确定性函数，随机性被移到了外部输入 $\epsilon$ 上，梯度可以正常反传。

> **Q: VAE 为什么生成模糊？**
> A: 根本原因是模型容量有限 + 高斯似然假设。一个 $z$ 可能对应多种不同的 $x$，但高斯只能输出一个均值，只好把多种可能性平均掉。改进方向：VQ-VAE（离散化）、VAE-GAN（加判别器）、更复杂的似然分布。

> **Q: DDPM 的训练目标是什么？**
> A: $L = \mathbb{E}\|\epsilon - \epsilon_\theta(x_t, t)\|^2$。随机选时间步 $t$，给干净图片加噪得到 $x_t$，让网络预测加了什么噪声。数学上是变分下界的简化版（去掉了与 $t$ 相关的权重），Ho et al. 发现这样训练效果反而更好。

> **Q: Diffusion 和 Score Matching 是什么关系？**
> A: 预测噪声 $\epsilon_\theta$ 等价于预测 score $\nabla_x \log p(x)$ 的缩放版：$s_\theta = -\epsilon_\theta / \sqrt{1-\bar{\alpha}_t}$。Yang Song 用 SDE 统一了两个框架——DDPM 和 NCSN 只是同一个 SDE 的不同离散化。

> **Q: Stable Diffusion 为什么要用 VAE？**
> A: 在像素空间做扩散太贵（786K 维）。VAE 把图片压缩到 64×64×4 的潜在空间（16K 维），扩散在潜在空间做完后再用 VAE Decoder 还原。这就是 Latent Diffusion Model 的核心思路。
