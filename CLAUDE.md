# Claude Code 项目规则

## 知识库维护说明

这是一个个人 AI 知识库项目，所有内容使用中文撰写，Markdown 格式。

### 操作权限

- 允许所有文件的读取、编辑、创建和搜索操作
- 允许使用 Glob、Grep、Read、Edit、Write 工具，无需额外确认
- 允许使用 Bash 执行 ls、mkdir 等文件管理命令

### 内容规范

- 所有知识库内容使用中文
- 使用清晰的 Markdown 格式
- 新增内容需更新对应目录的 README.md 索引
- 当目录下文件超过 15 个时，拆分为子目录
- 交叉引用使用相对路径
- 涉及特定版本的内容需标注版本和日期

### Git 规范

- 每次完成一组修改后，自动执行 git add + commit + push，不等用户要求
- commit message 用英文，简洁描述改动内容
- 每次修改后主动告知用户改了哪些文件和内容
